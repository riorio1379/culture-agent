# Culture Agent 🌸

日本文化・アジア文化を「正しい形」で世界に伝えるAIエージェント。
IBM AI Innovator Hackathon 応募に向けたMVP。

## 目的（成功条件）
ヨーロッパ人からの質問に対し、社内の文化資料を**自分で検索して**、
日本文化の背後にある価値観（謙虚さ・職人精神・マインドフルネスの仏教的ルーツ）を
敬意とともに、質問された言語で答えるAIエージェントを動かす。

## これは「AIを組み込んだシステム」である
| 構成要素 | 実装箇所 |
|---|---|
| ① LLMの組み込み | `agent.py` が `anthropic` SDK 経由で Claude API を呼ぶ |
| ② ツール使用（エージェント） | AIが `search_knowledge` ツールを自分で選んで呼ぶ agentic loop |
| ③ RAG（検索拡張生成） | `knowledge/` の資料を検索し、根拠に基づいて回答 |

「AIにコードを書かせた」のではなく、「コードの中でAIを部品として動かす」点が核心。

---

## 作成物の概要（IBM AI Innovator Hackathon 提出用）

### 何を作ったか・目的/背景
日本文化・アジア文化を「正しい形」で世界に伝えることを人生のミッションとしている。
ロンドンでの作務衣販売を通じ、日本文化が記号・高貴なものとしてしか伝わっていない
現実に直面した原体験から、AIで文化を正しく届ける手段としてこのエージェントを制作した。
ユーザーの質問に対し、社内の文化資料を自ら検索し、根拠に基づいて多言語で答える。

### 担当した役割（実装箇所・担当工程）
個人開発。コンセプト設計、アーキテクチャ選定（テキストRAG＋エージェント方式の決定）、
要件定義、知識ベースの設計・執筆、エージェントループとRAG検索ロジックの実装までを
一人で担当した。実装は生成AIを活用した開発手法（Claude Code）で進めた。
※「AIを用いた開発」自体が本コースの主題のため、AIと協働した開発工程を含む。

### 直面した課題と解決方法
- 課題1：LLMが資料にない事柄を「それらしく」答えてしまう（ハルシネーション）。
  → 解決：知識ベース検索（RAG）を必須化し、システムプロンプトで「資料にない事柄は
     推測せず正直に述べる」と制約。AIが回答前に必ず検索ツールを呼ぶ設計にした。
- 課題2：いつ検索すべきかをコード側で固定すると柔軟性が落ちる。
  → 解決：検索を「ツール」として定義し、AI自身が呼び出すか判断する agentic loop に
     変更。AIが必要に応じて自律的に知識ベースを参照する構造にした。

### 技術情報（モデル・API・アーキテクチャ・実装方法）
- 言語：Python
- モデル：Claude Opus 4.8（環境変数で Fable 5 へ切替可能な設計）
- API：Anthropic Messages API（`anthropic` SDK）
- アーキテクチャ：①LLM呼び出し ②Tool Use による agentic loop（AIが
  `search_knowledge` ツールを自律的に呼ぶ）③RAG（knowledge/ のmd資料を
  キーワードスコアリングで検索し、根拠としてLLMに渡す）
- 実装：`agent.py`（エージェント本体）＋ `knowledge/`（検索対象の文化資料）

## セットアップ

### 1. APIキーを取得する
1. https://console.anthropic.com にアクセスしてアカウント作成
2. 「API Keys」から新しいキーを発行（`sk-ant-...` で始まる文字列）
3. ターミナルで環境変数に設定:
   ```bash
   export ANTHROPIC_API_KEY="sk-ant-ここに自分のキー"
   ```
   ※毎回打つのが面倒なら `~/.zshrc` の末尾に上記1行を追記して `source ~/.zshrc`

### 2. ライブラリ（インストール済みなら不要）
```bash
pip3 install anthropic
```

### 3. 実行
```bash
cd ~/RIO/Workspace/Dev/culture-agent
python3 agent.py
```

#### モデルの切り替え（Opus 4.8 ⇄ Fable 5）
デフォルトは `claude-opus-4-8`（高性能・低コスト）。
最上位の **Fable 5**（約2倍コスト・より繊細な回答）で動かすには:
```bash
CULTURE_AGENT_MODEL=claude-fable-5 python3 agent.py
```
起動時に「🤖 使用モデル: ...」で確認できます。
普段のテストはOpus 4.8、本番デモやES用の見せ場だけFable 5、がおすすめ。

## 使い方の例
```
あなた> What is samue and why is it connected to Zen?
あなた> 日本の「謙虚さ」は西洋人にどう誤解されがちですか？
あなた> Is "zen" just a minimalist design style?
```
AIが自動で知識ベースを検索（🔎 が表示される）し、資料を根拠に回答します。
